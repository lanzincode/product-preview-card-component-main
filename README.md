### O que aprendi

Neste projeto, desenvolvi um **card de produto**, muito comum em interfaces de e-commerce, utilizando uma abordagem **desktop-first**, praticando:

- Estruturação de componentes reutilizáveis
- Organização visual de imagem, texto e preços
- Uso consistente de tipografia e espaçamento
- Criação de estados de hover
- Fidelidade ao design proposto pelo desafio

Esse desafio foi importante para consolidar **fundamentos de layout** e **componentização em CSS**.
